# Todo App (Node + HTML)

Simple todo app with an Express backend and static HTML/JS frontend.

Quick start

```bash
cd todo-app
npm install
npm start
# open http://localhost:3000
```

Files

- [todo-app/server.js](todo-app/server.js) — Express API and static file server
- [todo-app/public/index.html](todo-app/public/index.html) — frontend
